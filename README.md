# NGX-Investment-Report
This report  contains a data-driven investment analysis focused on seven stocks (MTNN, ZENITHB, GTCO, DANGCEM, SEPLAT, ARADEL, & PRESCO) on the Nigerian Exchange Group (NGX), designed to evaluate market trends, sector performance, and key equity opportunities for June 2026. The project pulls financial statements, trading data, and macroeconomic indicators to build valuation models, screen for overpriced stocks, and assess risk-adjusted returns across the telecom, banking, industrial goods, oil and gas, and agriculture sectors. My goal is to create a transparent, repeatable framework that combines fundamental analysis, financial modeling, and scenario testing to guide investment decisions on the NGX. The report includes a final investment brief summarizing buy and accumulation recommendations. It’s built for anyone looking to understand NGX dynamics, invest in the NGX exchange, backtest strategies, or use the empirical designs as a starting point for their own research.
